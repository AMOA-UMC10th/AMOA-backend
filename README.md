# AMOA-backend
> 네일아트 디자인 탐색부터 예약까지 연결하는 플랫폼 **AMOA**의 백엔드 서버입니다.

## 📌 프로젝트 소개

AMOA는 흩어져 있는 네일 디자인 정보를 한곳에서 탐색하고,
디자인, 가격, 위치를 비교하며 예약까지 이어질 수 있는 서비스를 목표로 합니다.

## 🛠 Tech Stack

### Backend
- Java 21
- Spring Boot 3.5
- Spring Data JPA
- Spring Security
- QueryDSL
- MySQL
- JWT
- Swagger(OpenAPI)

### Build Tool
- Gradle
